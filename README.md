4- Review and Approval Process

The LAMAS Scientific and Ethics Committee reviews all access requests. Evaluation criteria include:

- Compliance with IRB/CEP ethical approval

- Alignment with LGPD and GDPR data protection regulations

- Adequacy of privacy safeguards and risk mitigation

- Scientific validity, methodological soundness, and feasibility

- Institutional infrastructure for secure data handling

Applicants receive a written decision. The Committee may:

- Approve the request
- Request clarifications or modifications
- Reject proposals that do not meet ethical or legal criteria

5- Types of Access Granted

Depending on project scope, sensitivity, and privacy risk, approved applicants may receive one or more of the following:

- Curated derived datasets

- Annotations and structured metadata
- Non-identifiable imaging features
- Limited-access imaging subsets
- No access to raw full-resolution images if not strictly necessary

This tiered-access model maximizes scientific value while safeguarding patient privacy and institutional data governance.

6- Responsibilities of Data Recipients

All researchers granted access must adhere strictly to the Data Use Agreement:

- Store data in secure, access-controlled environments

- Never attempt patient re-identification

- Use data only for the approved project

- Do not share data with third parties

- Report any security breach or incident within 24 hours

- Destroy all data after project completion or upon DUA expiration
